# AGENTS.md

## Reglas del proyecto

- El objetivo es construir Realtime Voice Translator: una aplicación web para facilitar conversaciones presenciales entre dos personas que hablan idiomas diferentes usando un mismo dispositivo.
- El proyecto usa Spec-Driven Development; cada funcionalidad debe especificarse antes de implementarse.
- Lee `README.md`, `AGENTS.md` y la especificación correspondiente antes de implementar cambios.
- Implementa solo el alcance solicitado y no lo amplíes sin aprobación explícita.
- No agregues dependencias sin una justificación concreta.
- Mantén TypeScript en modo estricto.
- No expongas secretos, claves de API ni credenciales privadas al cliente.
- No almacenes audio ni conversaciones por defecto.
- Ejecuta las verificaciones relacionadas antes de finalizar una tarea.
- No hagas commits ni push.
